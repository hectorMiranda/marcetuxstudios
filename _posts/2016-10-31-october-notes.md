October was a big month personally. The ESP32 is in my hands and it's the right tool
for the home automation projects that the ESP8266 was making do for. Vue 2.0 is now
a framework I'd recommend for the right project. The API versioning post is the written
version of a conversation I've had three times this year. Queue state management is a
pattern I should have documented six months ago.

And I'm leaving JibJab. The decision feels right and the timing is deliberate. November
and December will be about closing things out well — the pipeline documentation, the
handoff notes, the things that only I know because only I built them. I've been bad
about documentation before and I don't want to hand the next person a pile of code
with no map. The blog will have more architecture retrospectives than new feature posts
for the rest of the year; there's value in writing down what you learned from something
before you move to the next one.

November goal: get the queue-driven workflow documentation actually into the repo, not
just the blog. Write the "how the transcoding pipeline works" doc that anyone who
touches it will need. ESP32 experiments alongside.
