April was a transition month in every sense. The GDPR work is the post I'm least
proud of writing — not because the work wasn't real, but because the compliance
exercise revealed some data hygiene shortcuts I should have caught earlier without
needing a regulation to prompt the audit. The API versioning post was more satisfying;
that's a solved problem with a right answer and I'm glad I documented the reasoning
rather than just the choice.

Blazor I'm watching. It went from "Steve Sanderson conference demo" to "public GitHub
repo with contributions" in the span of a month, and the engagement in the issues
tracker suggests Microsoft is taking it seriously. The WebAssembly runtime size will
come down; the tooling will mature. This is the same arc I watched with .NET Core from
2016 to 2018, and I know how to calibrate my patience for that arc now.

Start date at Corporate Travel Management is May 7. The last Go RN commit is done;
the handover doc is written and reviewed. I gave back the laptop on Thursday and then
immediately ordered a new mechanical keyboard because I apparently needed a thing to
look forward to.
