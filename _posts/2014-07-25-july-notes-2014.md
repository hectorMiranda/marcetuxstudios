Month two at Spark and the codebase is starting to have texture for me. The N+1 query
investigation was the kind of ticket that forces you to understand the data model quickly,
and the fix involved touching the model layer, the serializer, and the query all in one
PR. The Rails flow — change, test, deploy with Capistrano — is starting to feel natural
instead of like something I'm translating from C#.

Couchbase is the part I'm still learning. The document model is clear; the operational
side — monitoring for memory pressure, understanding when evictions start, reading the
bucket statistics in the admin UI — is the part that takes time to internalize. I have
an alert on bucket RAM usage and I've been watching the eviction rate during peak hours.
Nothing alarming yet, but I want to understand the headroom before we're in it.

The ESP8266 is sitting on my kitchen table next to the Arduino. I got it to join the
WiFi network and POST a JSON payload to a small HTTP server I ran on the laptop. The
latency is good, the power consumption is low, and the price is absurd for what it does.
I ordered five more. Future-me is going to wire the whole apartment with these.
