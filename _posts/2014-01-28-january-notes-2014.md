January under the Verizon roof, and the building is the same building. The acquisition
paperwork is apparently still grinding through regulatory review, so we're in a period
where everything is officially normal and nobody quite believes it. I'm using the
stability window to actually fix things: Gulp replaced Grunt, the customer controller is
sixty lines instead of three hundred, and the response cache means the reporting endpoints
no longer fall over when the morning shift opens twelve browser tabs at once.

The 3D printer landed and taught me that all the things that seemed like they'd be
the hard part — slicing software, filament tension, temperature profiles — are actually
tractable with a few hours of trial prints. The hard part is the bed. Always the bed.
The Raspberry Pi case came out passable; a replacement bracket for a broken fan mount
in my server rack came out perfect on the second try. The machine is already useful.

February is about the Angular router rewrite I've been avoiding. The current routing
setup was wired in a hurry and doesn't compose well with the new lazy-loading pattern
I want. And I want to print something larger — the calibration cube era is over.
