The year started with a Grunt build and an Angular app I didn't fully trust, and ended
with a Gulp build, typed Angular services, a Web API 2 REST surface behind OWIN/Katana,
Redis for sessions and pub/sub and atomic Lua operations, SignalR with a Redis backplane,
E2E tests in Protractor, async controllers, HMAC-signed API clients, and a bench
logger board that made it to v1.1 without any bodge wires. The PCB checklist from a
mirrored FT232 footprint is more useful than the bad board it corrected.

The external events that mattered technically: Docker showed up in March and is a Linux-
only curiosity on our Windows stack; Bootstrap 3 arrived in August with the mobile-first
grid that's now the default for anything new; the Snowden revelations in June pushed the
HTTPS audit forward by at least a year. Bitcoin's run from $100 to past $1,000 is the
most entertaining real-time CDN caching case study I've watched. And Verizon announced
the acquisition in December, which is the kind of external event that makes you value
the skills you've built over the year more than the org structure that contains them.

The things that stayed the same: the code review discipline, the habit of reading the
plan before adding an index, the preference for boring at the boundary, and the
electronics bench that's running whether or not there's anything particularly urgent
to measure. 2014: async queue for report generation, Docker when it reaches Windows,
and let's see what Verizon actually does with EdgeCast.
