October was about going deeper rather than going wide. Service objects, cursor
pagination, CloudWatch custom metrics — all of these are improvements to things that
were working but not well. The kind of refactor that doesn't generate excitement but
makes the codebase easier to work in six months from now. I find I like that kind of
work more than I expected to.

The home sensor network has data now — six weeks of temperature readings and door events.
The patterns are boring in the expected ways: the apartment cools at night, warms in the
afternoon, the door opens most often between 6-8am and 6-8pm. The interesting one is
the temperature differential between the bedroom and living room: consistently three
degrees warmer in the bedroom during afternoons because of direct sun through the blinds.
I didn't know that before the sensor told me.

November is going to be interesting. There's a rumor about Microsoft open-sourcing .NET,
which I'll believe when I see it. And I want to clean up the home sensor API — the
Sinatra code is functional but messy, and I've been writing enough Rails at work that
I know how it should be structured now.
