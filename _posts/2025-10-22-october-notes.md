The power monitor board is at the fab, and the first real fall cooldown arrived this
week — studio temperature is stable in the mid-60s now without any active management.
The CO2 sensor network has been running for six months and the data is genuinely useful:
I can see the CO2 correlate with the number of hours I've spent in the space, which is
a proxy for "how hard was I working today" that turns out to be more honest than
self-reporting.

The consulting work has shifted slightly this month toward the production-side concerns
— cost attribution, reliability, observability — rather than the initial build-out work.
Teams that shipped in 2024 are now in maintenance and optimization mode, which is a
different and honestly more interesting set of problems. The event-driven pipeline post
came from one of those engagements.

Agentic code review is something I've quietly been using for months and finally wrote
down. The teams I work with that have it set up don't want to turn it off, which is the
best signal that a tool is actually useful rather than just impressive in a demo.
