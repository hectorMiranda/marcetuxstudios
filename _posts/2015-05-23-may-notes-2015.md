May was a performance month. Elasticsearch replacing Postgres ILIKE queries, N+1
queries killed with Bullet and `includes`, the memoization bug found and fixed, Route 53
weighted routing making a risky algorithm rollout into a boring one. Each of these was
a known problem; the month was about having enough runway between features to actually
close them.

The professional conversation I'm having with myself is in the previous post. Putting
it down here so the record is honest: this is the moment I started actively looking.
Not urgently, not unhappily, but deliberately. The media-and-video direction is where
the interesting engineering problems seem to cluster from where I'm sitting.

Pi 2 as a build server for the ESP8266 project is working, which is satisfying in a way
that's hard to explain — a $35 computer running CI for a $3 sensor node, all connected
through a MQTT broker, all visible in Grafana. The stack is ridiculous in scale and
exactly right in concept. That's the kind of thing I want more of.
