June was the transition month in full. I left a CDN I understood deeply and joined a
consumer product I understand partially, which is exactly the level of discomfort
that produces growth. The Rails codebase is real work — not complex in a bad way,
just rich with domain context that takes time to acquire. I'm reading more code and
writing less than I usually do, which is the right posture for the first month.

Docker 1.0 landed this week and I've been running it in the evenings. The container
model is clicking for me in a way that Vagrant's VM model never quite did — the
lightweight nature of it changes what you're willing to containerize. I want to explore
using it for local development at the new job, but first I need to understand what the
existing development environment actually is.

The Santa Monica apartment is starting to feel like a home. The Printrbot is printing
again after surviving the move in a laptop bag. Printed a cable organizer for the desk
that actually works better than the one I'd been using. Next: design a case for a small
ESP8266 module I picked up — the chip showed up in some forum posts I've been reading
and the idea of a $4 wifi module is hard to ignore.
