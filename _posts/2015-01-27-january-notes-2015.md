January was about closing the loop on things started at the end of last year. Couchbase
view indexes are working now, though the development feedback loop is still slower than
I'd like — next month I want to wire a small local dataset fixture so I can iterate
faster. The SQS queue decoupled the match workers and the first evening-peak load test
didn't produce the cascading timeouts we'd been seeing, so that's a win.

The home bench got interesting: Pi 2 is running Mosquitto as an MQTT broker, and the
first ESP8266 node is publishing temperature every 30 seconds without complaint. The
economics of that combination are hard to argue with — serious Linux box as the hub,
disposable WiFi nodes in every corner. The stack is almost exactly what I sketched in a
notebook last spring.

Watching React Native from a distance. The bridge-to-UIKit approach is architecturally
honest in a way that the WebView-wrapper era never quite managed, and if it ships the
way Facebook is showing it the mobile conversation at Spark gets more interesting. For
now: Babel in the Gulp pipeline, ES6 classes in the matching module, and a Capistrano
deploy that doesn't require an SSH session to finish correctly.
