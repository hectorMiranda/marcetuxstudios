# marcetux studios

A personal tech blog by **marcetux** — notes from the trenches of building web
platforms, APIs, and real-time systems, with the occasional detour into
electronics and whatever I'm soldering on the bench this week.

It's plain Markdown and a static site generator. No tracking, no fluff — just
things I learned the hard way and want to remember.

## Running locally

```bash
gem install jekyll bundler
jekyll serve
# http://localhost:4000
```

Posts live in `_posts/`. New writing shows up newest-first on the home page.
