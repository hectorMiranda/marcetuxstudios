# Examples

Runnable bits that go with the posts. Organized by year. Most are deliberately
small — enough to try the idea, not a framework.

## 2012
- `totp/Totp.cs` — RFC 6238 TOTP generator (goes with the two-factor post).
- `arduino/dht11_logger.ino` + `arduino/logger.py` — the "Arduino senses, Pi
  remembers" sensor pipeline.
- `sql/covering_index.sql` — turning a reporting scan into a seek.
- `knockout/dashboard.html` — a tiny MVVM dashboard.

Each example is self-contained; see the comment header in each file for how to run it.
